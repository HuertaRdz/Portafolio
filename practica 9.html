Unidades de Medida en CSS

Unidades Absolutas

Las unidades absolutas tienen un tamaño fijo que no cambia en relación con otros elementos.

Píxeles (px)

200px × 100px

1px = 1/96 de pulgada. Ideal para bordes y sombras.

Pulgadas (in)

2in × 1in

1in = 2.54cm. Usado principalmente para medios impresos.

Centímetros (cm)

5cm × 2cm

Basado en el sistema métrico. Útil para diseños de impresión.

Unidades Relativas

Las unidades relativas dependen del contexto (elemento padre o viewport).

em

Relativo al font-size del elemento padre.

rem

Relativo al font-size del elemento raíz (html).

Porcentaje (%)

50% del ancho del contenedor padre.

Útil para crear diseños fluidos.

Viewport (vw/vh)

50vw × 20vh

1vw = 1% del ancho del viewport. 1vh = 1% del alto.

Recomendaciones de Uso

Cuándo usar unidades absolutas

  • Bordes y sombras (px)
  • Diseños para impresión (in, cm, mm, pt)
  • Cuando se necesita precisión absoluta
  • Elementos que no deben escalar

Cuándo usar unidades relativas

  • Diseños responsive (rem, vw, vh, %)
  • Tamaños de texto (rem para escalado consistente)
  • Espaciado entre elementos (em, rem)
  • Diseños fluidos y flexibles
